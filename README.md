# WAUVSim

Simulation repo for WAUV.

Instructions:
- Open a terminal in Ubuntu 22.04 (with WSL2)
- Clone this repo in a directory named 'WAUV'
- In the bash terminal, run the command './install.sh'
- To launch the simulation, navigate to '~/WAUV/WAUVSim/src/wauv_sim/launch' and enter the command 'ros2 launch bluerov_sim.launch.py'
